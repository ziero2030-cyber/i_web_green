# 🛋️ Overseas Furniture Brand Shopping Mall (i-web Project)

i-web 플랫폼을 활용하여 제작한 **해외 브랜드 가구 쇼핑몰 클론 및 리디자인 프로젝트**입니다. 본 프로젝트는 세련되고 감각적인 가구 브랜드의 이커머스 플랫폼을 모티브로 하여 구현되었습니다.

---

## 📌 프로젝트 개요
* **제작 플랫폼:** i-web (아이웹)
* **프로젝트 유형:** 반응형 웹사이트 / 이커머스 쇼핑몰
* **주요 타겟:** 감각적인 인테리어와 해외 브랜드 가구에 관심이 높은 사용자층
* **핵심 컨셉:** 모던, 미니멀리즘, 하이엔드 라이프스타일 제안

---

## ✨ 주요 기능 및 특징
* **반응형 웹 디자인 (Responsive Design):** 데스크탑, 태블릿, 모바일 등 다양한 디바이스 환경에 최적화된 레이아웃 제공
* **브랜드 큐레이션 존:** 단순 상품 나열이 아닌, 브랜드 고유의 가치와 스토리텔링을 강조한 메인 배너 및 기획전 섹션 구성
* **직관적인 UX/UI:** 상품 탐색부터 상세 페이지까지 매끄럽게 이어지는 미니멀하고 직관적인 인터페이스
* **이커머스 기능 구현:** i-web 솔루션을 활용한 카테고리 분류, 상품 상세 정보 제공 및 쇼핑 편의 기능 연동

---

## 🛠️ 사용 기술 및 도구
* **Platform:** i-web Engine
* **Frontend Customizing:** HTML5, CSS3 (Bootstrap 기반 레이아웃 최적화)
* **Design Concept:** Pastel & Modern Tones, Clean Minimal Layout
* **Version Control:** Git / GitHub
